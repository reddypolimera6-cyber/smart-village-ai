# 🌾 SmartVillage AI

AI-powered rural smart village application built using Python and Streamlit.

---

## 🚀 Features

* 🤖 AI Crop Advisory (upload crop images)
* 🚜 Equipment Booking (Drone, Tractor)
* 🌦 Weather Information
* 👨‍🌾 Community Chat

---

## 🛠 Tech Stack

* Python
* Streamlit

---

## ▶️ How to Run

1. Install Streamlit:
   pip install streamlit

2. Run the app:
   streamlit run app.py

---

## 🎯 Goal

To help rural farmers improve productivity using AI and shared resources.
